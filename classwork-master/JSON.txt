json:
	{} - объявление объекта;
	[] - объявление массива;
	123 - объявление чилового значения;
	"hello" - объявление строкового значения;
	true,false - объявление логического значения;
	
Примеры:
	{ 
		"name": "Vasya",
		"age": 99,
		"isMarried": true,
		"favouriteDrinks": ["Cola","Sprite","Pepsi"],
		"passport": {
			"number": "KB9379992",
			"expireDate": "06/06/2025"
		}
	}
